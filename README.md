# Sistema de PDV Full Stack 🚀

Este é um sistema de Ponto de Venda (PDV) completo, desenvolvido para facilitar a gestão de vendas e estoque de pequenos negócios.

## 🔗 Link do Projeto Online
[Acesse aqui o sistema rodando no Render](https://meu-pdv-a51l.onrender.com)

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** JavaScript (Node.js)
- **Framework Web:** Express
- **Banco de Dados:** MongoDB Atlas (Nuvem)
- **Hospedagem:** Render.com

## ⚙️ Funcionalidades
- Cadastro de produtos e controle de estoque.
- Registro de vendas em tempo real.
- Banco de dados resiliente (Migrado de SQLite para MongoDB para persistência de dados).

## 🚀 Como rodar o projeto localmente
1. Clone o repositório: `git clone https://github.com/Wolly06/NOME-DO-SEU-REPO.git`
2. Instale as dependências: `npm install`
3. Inicie o servidor: `npm start`
